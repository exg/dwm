[dwm](https://dwm.suckless.org/) fork with the following additional features:

- multiple keymaps
- run-or-raise action
- per tag layout configuration (views with two or more tags share the same configuration) - based on
  [pertag patch](https://dwm.suckless.org/patches/pertag/)
- restart action - based on
  [restartsig patch](https://dwm.suckless.org/patches/restartsig/)
- tab bar - based on
  [tab patch](https://dwm.suckless.org/patches/tab/)
